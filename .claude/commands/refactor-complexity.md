# Refactor Complexity Analysis

Analyze the provided Flutter/Dart file for cyclomatic complexity and propose a comprehensive refactoring strategy to improve code maintainability, testability, and readability.

The skill will:
- Calculate cyclomatic complexity for all methods
- Identify code duplication
- Propose service extraction strategy
- Recommend widget extraction
- Provide implementation roadmap
- Include testing strategy

**Usage**: `/refactor-complexity [filepath]`

**Example**: `/refactor-complexity lib/screens/chat_ai_screen.dart`
