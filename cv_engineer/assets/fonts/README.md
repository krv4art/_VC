# Fonts Directory

This directory should contain the font files referenced in pubspec.yaml.

## Required Fonts

### Roboto
- Roboto-Regular.ttf
- Roboto-Bold.ttf

### Merriweather
- Merriweather-Regular.ttf
- Merriweather-Bold.ttf

## Download Instructions

1. **Roboto**: Download from [Google Fonts](https://fonts.google.com/specimen/Roboto)
2. **Merriweather**: Download from [Google Fonts](https://fonts.google.com/specimen/Merriweather)

Extract the .ttf files and place them in this directory following this structure:
```
assets/fonts/
├── Roboto-Regular.ttf
├── Roboto-Bold.ttf
├── Merriweather-Regular.ttf
└── Merriweather-Bold.ttf
```

Alternatively, the app will fallback to system fonts if these files are not present.
