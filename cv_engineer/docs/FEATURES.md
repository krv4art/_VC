## Resume Editing

### Experience Editor

The Experience Editor provides a comprehensive interface for managing your work history:

**Features:**
- **Job Details**: Title, company name, and location
- **Date Management**: Start and end dates with visual date pickers
- **Current Job**: Toggle to indicate ongoing employment
- **Description**: Free-form text area for role overview
- **Responsibilities**: Multiple bullet points for key achievements
- **Duration Calculation**: Automatic calculation and display of employment duration

**How to Use:**
1. Navigate to Resume Editor → Experience
2. Tap the floating action button (+) to add new experience
3. Fill in the required fields (marked with *)
4. Add responsibilities using the "Add" button
5. Tap the checkmark to save
6. Edit or delete existing entries using the action buttons

**Tips:**
- Use action verbs in responsibilities (e.g., "Developed", "Managed", "Led")
- Keep descriptions concise and focused on achievements
- Add 3-5 key responsibilities per role

### Education Editor

The Education Editor helps you document your academic background:

**Features:**
- **Degree Information**: Degree name, institution, and location
- **Date Tracking**: Start and end dates with current studies option
- **GPA**: Optional field for grade point average
- **Achievements**: Multiple entries for honors, awards, and accomplishments
- **Description**: Additional details about your studies

**How to Use:**
1. Navigate to Resume Editor → Education
2. Add new education entry
3. Enter degree and institution (required)
4. Set dates and mark if currently studying
5. Add GPA if applicable (e.g., "3.8/4.0")
6. List achievements and honors

### Skills Editor

The Skills Editor features advanced categorization and proficiency tracking:

**Features:**
- **Skill Categories**: Technical, Programming Languages, Frameworks, Tools, Soft Skills, etc.
- **Proficiency Levels**:
  - Beginner (25%)
  - Intermediate (50%)
  - Advanced (75%)
  - Expert (100%)
- **Visual Indicators**: Progress bars showing proficiency
- **Grouped Display**: Skills organized by category

**How to Use:**
1. Navigate to Resume Editor → Skills
2. Tap (+) to add a new skill
3. Enter skill name (autocomplete suggestions available)
4. Select or create a category
5. Choose proficiency level
6. Skills are automatically grouped by category

**Preset Categories:**
- Technical Skills
- Programming Languages
- Frameworks & Libraries
- Tools & Platforms
- Soft Skills
- Languages
- Other

### Languages Editor

The Languages Editor follows CEFR (Common European Framework of Reference) standards:

**Features:**
- **Language Selection**: Autocomplete with 19 common languages
- **CEFR Levels**:
  - A1 - Beginner
  - A2 - Elementary
  - B1 - Intermediate
  - B2 - Upper Intermediate
  - C1 - Advanced
  - C2 - Proficient/Native
- **Professional Display**: Badge-style indicators with level codes

**How to Use:**
1. Navigate to Resume Editor → Languages
2. Add language (autocomplete helps with spelling)
3. Select appropriate CEFR level
4. Languages display with proficiency badges

**Common Languages Included:**
English, Spanish, French, German, Italian, Portuguese, Russian, Chinese, Japanese, Korean, Arabic, Hindi, Polish, Turkish, Dutch, Swedish, Norwegian, Danish

## PDF Export

### Export Options

CV Engineer provides three ways to export your resume:

**1. Save to Device**
- Tap the PDF icon in Preview screen
- PDF saves to Documents directory
- File name includes timestamp
- Notification shows save location

**2. Share**
- Tap the Share icon
- System share dialog appears
- Share via email, messaging, cloud storage
- Custom filename: `[Your_Name]_resume.pdf`

**3. Print**
- Tap the Print icon
- System print dialog appears
- Print directly or save as PDF
- Preview before printing

### PDF Features

**Professional Formatting:**
- A4 page size
- Respects your font size settings (8-16pt)
- Customizable margins (10-40pt)
- Roboto font family
- Color-coded section headers
- Proper spacing and alignment

**Included Sections:**
- Header with contact information
- Professional summary
- Work experience with bullet points
- Education with achievements
- Skills grouped by category
- Languages with CEFR levels

## AI Assistant

### Content Improvement

The AI Assistant provides intelligent content enhancement:

**Available Functions:**

**1. Improve Text**
```
Input: "made the website better"
Output: "Enhanced website performance and user experience through optimization techniques"
```

**2. Generate Summary**
- Analyzes your experience and skills
- Creates professional summary
- Tailored to target role (optional)

**3. Suggest Skills**
```
Job Title: "Software Engineer"
Suggestions: Problem Solving, Git, Agile Methodologies, Testing, Code Review
```

**4. Fix Grammar**
- Corrects spelling errors
- Fixes punctuation
- Ensures proper capitalization

**5. Make Professional**
- Removes casual language
- Uses professional terminology
- Improves clarity and impact

### Fallback Mode

When AI service is unavailable, the app uses smart fallback methods:
- Basic text improvements (capitalization, punctuation)
- Template-based summaries
- Preset skill suggestions by job category
- Simple grammar fixes

## Photo Management

### Adding a Photo

**Methods:**
1. **Gallery**: Choose from existing photos
2. **Camera**: Take a new photo

**Process:**
1. Navigate to Personal Information
2. Tap photo placeholder
3. Select source (Gallery or Camera)
4. Photo automatically:
   - Compresses to 85% quality
   - Resizes to max 800x800px
   - Saves to app documents
5. Delete by tapping photo and confirming

**Permissions:**
- Android: Requests storage/camera permissions
- iOS: Requests photo library/camera access
- Web: Uses browser file picker

### Photo Guidelines

**Best Practices:**
- Use a professional headshot
- Ensure good lighting
- Plain background preferred
- Face clearly visible
- Appropriate professional attire

## Tips & Best Practices

### General Resume Tips

**Length:**
- Entry-level: 1 page
- Mid-career: 1-2 pages
- Senior/Executive: 2 pages

**Content:**
- Focus on achievements, not responsibilities
- Use metrics when possible (e.g., "Increased sales by 30%")
- Tailor resume to each job application
- Keep language professional and concise

### Using the App

**Workflow:**
1. Start with Personal Information
2. Add Work Experience (most recent first)
3. Add Education
4. List relevant Skills
5. Add Languages (if applicable)
6. Review and adjust formatting
7. Preview before exporting
8. Export as PDF

**Before Exporting:**
- Check completeness indicator (aim for 100%)
- Preview resume appearance
- Adjust font size if needed
- Modify margins to fit content

### Interview Preparation

Use the Interview Questions feature:
- Navigate to Home → Interview Prep
- Review common questions by category
- Prepare answers using STAR method:
  - **S**ituation
  - **T**ask
  - **A**ction
  - **R**esult

## Keyboard Shortcuts

When using web version:
- `Tab` - Navigate between fields
- `Enter` - Save form (when on save button)
- `Esc` - Close dialog/cancel

## Troubleshooting

### PDF Export Issues
- **PDF not saving**: Check device storage permissions
- **PDF looks different**: Ensure preview matches before export
- **Share not working**: Check app has share permissions

### Photo Upload Issues
- **Camera not opening**: Grant camera permission in settings
- **Photo not saving**: Check storage permission
- **Photo quality low**: Ensure original photo is high quality

### General Issues
- **App crashes**: Restart app, data is auto-saved
- **Changes not saving**: Wait for save confirmation
- **Preview not updating**: Navigate away and back to preview
