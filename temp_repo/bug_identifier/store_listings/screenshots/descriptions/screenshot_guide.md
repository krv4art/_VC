# Screenshot Guide for AI Cosmetic Scanner

## Required Screenshots

### Google Play Store
- **Phone screenshots**: 2-8 images (minimum 2)
- **Tablet screenshots**: 2-8 images (optional but recommended)
- **Size**: Minimum 320px, Maximum 3840px
- **Aspect ratio**: 16:9 or 9:16
- **Format**: PNG or JPG (no alpha channel)

### Apple App Store
- **iPhone 6.7"** (iPhone 14 Pro Max): 1290 x 2796 px (required)
- **iPhone 6.5"** (iPhone 11 Pro Max): 1242 x 2688 px (required)
- **iPhone 5.5"** (iPhone 8 Plus): 1242 x 2208 px
- **iPad Pro 12.9"**: 2048 x 2732 px (if supporting iPad)
- **Format**: PNG or JPG (no transparency)

## Design Tips

1. **Text Overlays**:
   - Use large, readable fonts
   - High contrast (white text on dark overlay)
   - Keep text concise (5-7 words max)
   - Position at top or bottom 1/3 of screen

2. **Highlight Key Features**:
   - Use circles/arrows to point out important UI elements
   - Add subtle shadows or glows to important buttons
   - Use brand colors consistently

3. **Real Data**:
   - Use actual cosmetic product names
   - Show realistic ingredient lists
   - Display genuine analysis results
   - Make it relatable

4. **Backgrounds**:
   - Clean, minimalist backgrounds
   - Gradient overlays work well
   - Consider your brand colors
   - Maintain consistency across all screenshots

5. **Localization**:
   - Create separate screenshots for each language
   - Ensure text overlays are translated
   - UI elements should be in target language
   - Check for text overflow issues
